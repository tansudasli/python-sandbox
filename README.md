# python-sandbox
Python sandbox w/ core capabilities 

### key concepts
- data structures (list [], tuple (), range, dict {k:v}, set {})
- higher order functions (=hof)
   - map, filter, sort -inplace, sorted ...
   - comprehensions = one-line = kinda-hof 
   - operator = built-in-functions-as-lambdas (min, max, sum...)
   - itertools (accumulate, groupby, combination, count, cycle, repeat, batched, chain ...)
   - functools (reduce)
   - min, max, len
- iteration

- generators (...for..if..), (yield) -> next
- decorators
- closures
- callbacks



### key libraries
* pandas =  file (read_csv, pd.Series(), to_numpy, tolist), data wrangler (filter, groupby, stack/unstack)
* numpy = scalar/vector/tensor, random, +distributions
* scipy = sparse, optimizers, significance tests, spatial, graph ...
* matplotlib 
* ml models
   * linear regression w/ scipy
   * natural language processing w/ nlptk
   * n-gram analysis w/ nlptk
   * sentiment analysis w/ nlptk and scikit-learn
